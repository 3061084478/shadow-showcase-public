# 前端设计与页面分区

## 视觉方向

Shadow 的业务页采用统一的深蓝夜景与暗色玻璃风格：

- 深蓝背景与冷色微光
- 低饱和、高对比的文字层级
- 面板、筛选器、输入框与状态色统一收口到语义 token

## 页面结构

## 登录页

- 二维码登录入口
- 本地 API 启动状态
- 登录态轮询与结果反馈

## 首页

- 好友列表与最近使用好友
- unknown 数据贡献授权入口
- 刷新归档按钮
- 页面跳转入口

## 聊天记录查询页

- 左侧筛选面板
- 时间、发送方、消息类型、查询方式切换
- 右侧消息结果卡片
- 顶部命中数与时间线说明

## 音乐关系页

- 单好友分析
- 全局音乐社交画像
- 轨道、热力图、时间河流、共同音乐世界等沉浸式可视化模块

## 影子歌单页

- 当前目标歌单
- 切换歌单 / 新建歌单双面板
- 候选歌曲查询
- 当前歌单内容与候选结果区
- 生成影子歌单主操作
