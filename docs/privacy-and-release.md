# 隐私、安全与发布方式

## 隐私边界

Shadow 的目标不是把所有内容搬到云上，而是在保证本地体验的前提下，只把真正需要的最小公共数据送到云端。

## unknown 贡献默认关闭

- 首次使用时默认不授权
- 开启后只上传：
  - 歌名
  - 歌手数组
  - 专辑名
- 不上传：
  - 网易云 Cookie
  - 聊天记录原文
  - 好友 UID
  - 完整歌单
  - GPT 内容

## 基础安全策略

- unknown 提交接口限流
- 云端按标准化 key 去重
- 管理接口要求管理员令牌
- 日志避免记录敏感正文
- 客户端所有本地运行产物写回自身数据目录
