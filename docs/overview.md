# Shadow 项目总览

## 这是什么

Shadow 是一个以“音乐社交痕迹整理”为主题的本地 Web 应用。它从网易云音乐的好友聊天与歌曲分享记录出发，把原始消息整理成可查询、可分析、可继续生成歌单的结构化结果。

## 解决的问题

- 用户很难长期回看与好友之间的音乐互动
- 聊天里分享过的歌经常散落在时间线里，缺乏统一归档
- 已有分类映射无法覆盖所有歌曲，unknown 数据需要持续补齐
- 想做可交付产品，但又不希望把聊天内容、Cookie、好友 UID 全部上传到云端

## 核心体验

- 登录网易云后，本地拉取好友与消息数据
- 归档聊天中的歌曲分享内容
- 以好友为维度查看聊天记录、音乐关系和歌单候选
- 建立专属好友的私信分享歌单
- 对无法识别类型的歌曲，仅上传最小三字段（歌曲、歌手、专辑）参与公共映射完善
- 客户端自动获取最新 mapping delta，逐步提升本地识别能力

## 项目边界

- 不是完整 SaaS
- 不做云端聊天存储
- 不做用户账号体系
- 不把完整歌单、聊天原文、Cookie 上传到云端

## 当前成品形态

- 本地前端：React + TypeScript + Vite
- 本地后端：Python + SQLite
- 本地 API 网关：NeteaseCloudMusicApi
- 云端：CloudBase unknown 收集与 mapping delta 发布
- 发行方式：Windows 便携包
